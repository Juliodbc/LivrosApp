## 📖 Descrição do Projeto

O **LivrosApp** é uma aplicação front-end desenvolvida com **Ionic + Vue + TypeScript**, com o objetivo de simular uma plataforma de catálogo de livros.

O sistema permite ao usuário navegar entre diferentes telas, visualizar livros disponíveis, acessar detalhes de cada obra e interagir com funcionalidades como favoritos — tudo isso utilizando um banco de dados fictício.

O foco do projeto está na construção de uma interface moderna, responsiva e com navegação fluida, seguindo boas práticas de desenvolvimento front-end.

---

## 🧩 Funcionalidades

* Navegação entre telas utilizando Vue Router
* Listagem de livros
* Visualização de detalhes de cada livro
* Marcar/desmarcar livros como favoritos
* Interface responsiva (mobile e desktop)
* Simulação de autenticação (login/logout)
* Recuperação de senha (visual)

---

## 📱 Telas Desenvolvidas

* 🔐 **Login**

  * Acesso ao sistema
  * Navegação para cadastro e recuperação de senha

* 📝 **Cadastro (Register)**

  * Criação de nova conta (simulação)

* 🔁 **Recuperação de Senha (Recover)**

  * Envio de e-mail (simulado)

* 🏠 **Home (Biblioteca)**

  * Listagem de livros
  * Filtro de favoritos
  * Navegação para detalhes
  * Logout

* 📖 **Detalhes do Livro (BookDetail)**

  * Exibição de informações completas
  * Ação de favoritar/desfavoritar
  * Botão de retorno
